\doxysubsection{Detailed Description}
Provides a tool to build maps based on a image. 

Each pixel of the image can be converted into a block of its original color, a block with an especific \mbox{\hyperlink{struct_g_l_tech2_1_1_texture}{Texture}} or nothing.
