Gets and sets the angle, in degrees, of the vector, approximately preserving it\textquotesingle{}s angle. 

Getting angle property will always return a value greater than -\/90 and less or equal to 270 due to performanceoptimizations and can be changed in future patches. 

Modifying the angle of a vector is equivalent to creating a new vector with the same module and the specified angle, which can be slow if done many times and, due to floating-\/point accuracy problems

, can distort the original vector\textquotesingle{}s module if changed many times. If you wish to constantly change the angle of a vector, consider holding the module value and constantly creating new vectors by module and angle via \mbox{\hyperlink{struct_g_l_tech2_1_1_vector_a5e4956f0fec3061aead351430cc0ade7}{Vector.\+From\+Angle(float, float)}} or new Vector(flaot). \mbox{\Hypertarget{struct_g_l_tech2_1_1_vector_aaba676429c077e27c5ce937bf6890d8b}\label{struct_g_l_tech2_1_1_vector_aaba676429c077e27c5ce937bf6890d8b}}
