◆ Angle

float GLTech2.Vector.Angle
getset

Gets and sets the angle, in degrees, of the vector, approximately preserving it's angle.

Getting angle property will always return a value greater than -90 and less or equal to 270 due to performanceoptimizations and can be changed in future patches.

Modifying the angle of a vector is equivalent to creating a new vector with the same module and the specified angle, which can be slow if done many times and, due to floating-point accuracy problems

, can distort the original vector's module if changed many times. If you wish to constantly change the angle of a vector, consider holding the module value and constantly creating new vectors by module and angle via Vector.FromAngle(float, float) or new Vector(flaot).
